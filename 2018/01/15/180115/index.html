风险管理与内部控制课程笔记

发表于 2018-01-15 | 分类于 学习
2017-2018第一学期风险管理与控制课程笔记

李心合部分

一、内控背景

  • 政治背景：政治生态变化——反腐败——内控是发腐的基础性工程。
  • 经营环境：宏观经济环境、技术环境、商业模式创新 ——高风险经营时代。
  • 风险事件：国有企业的重大资产损失事件：省属国企的系列事件；上市公司的财务舞弊案件：舜天船舶事件；私营企业的经营失败案件：雨润事件。

二、对“内部控制”的理解

《基本规范》：“内部控制，是由企业董事会、监事会、经理层和全体员工实施的、旨在实现控制目标的过程。”

我的理解：内控是通过流程（“形”）实现制衡（“神或魂”），达到有效防控风险“目标”的过程。

（一）内控之“形”：流程

基本流程或流程内核：提——审——决

  • “提”：提出、申请。
  • “审”：审核、审定。
  • “决”：决定、审批。
  • 附加流程：评审、鉴定、审计等。

上市公司财报签字：

  • 财会记录负责人
  • 财务负责人：总会计师、财务总监等，分管财务的总经理、副总裁（这两种多见），董事长直管财务（少见）
  • 法定代表人：董事长或总经理

流程：编制人编制 -> 财会负责人一审 -> 提交给财务负责人二审 -> （注会的社会审计） -> 提交给董事会审议 -> 法定代表人签字 —> 对外披露
注册会计师在审计报告而非财务报告上签字

  • 流程的合规性：公司法、三重一大制度、监管法规等。
  • 流程的效率性：避免控制过度、协调控制与效率关系。
  • 流程的风险性：避免控制不足、有效防控风险（省属企业分权管理案例）。
  • 流程的有效性：避免程序化和走过场（省属企业案例）。
  • 流程的经济性：避免控制成本过高——流程长、高成本控制工具（先进的平衡计分卡等）或程序（集体决策）。

控制过度和控制不足都要避免
控制过度会造成

  1. 效率低
  2. 成本高
  3. 效果差（节点越多，搭便车现象越严重）
  4. 责任难追究

（1）“提”（申请）

谁提出？

  • 个人（涉及个人事项）提出
  • 机构或部门（公司事项）提出［机构——董事会、总经理办公会等，部门—经办部门或归口部门］

何种方式提出

  • 表（申请表）：内部的
  • 单（报销单等） ：收据发票，是外部来的
  • 报告（申请报告）：注意与管理方案相关

什么时间提出

  • 有时间要求的
  • 无时间要求的

（2）“审”（审核）

谁审

  • 一审（主管或专职审核岗位）
  • 二审（部门负责人）
  • 三审（分管领导）
  • 会审会签

审什么

  • 业务审核（业务的合规性、合理性）
  • 财务审核（形式审核—单证的完整性和完备性，实质审核——单证的真实性、合规性和财务收支的合理性）完整完备、真实合理合规

收据，单位的自制收据不可报销，财政机关颁发的行政事业费收据可以报销

审的时间要求

  • 流程中应当有时间规定

（3）“决”（决定、审定、审批等）

谁有权决定或审批

  • 集体决策（党委会、董事会、办公会、职代会等）
  • 个人审批（董事长、总经理、分管领导、部门负责人）

党委会：中层人事任免（重要干部任免）

董事会：见公司法

  1. 审批预算
  2. 大额长期贷款和大额资金调度
  3. 年度经营计划（单项不一定董事会批）
  4. 任免财务负责人等

总经理办公会：不上报给董事会的重大事项

职代会（不是终极审批，要再提交给董事会审批）

  1. 职工身份（如有终身制变为劳动合同制）
  2. 职工薪酬改革方案
  3. 职工代表（员工董事和员工监事）
  4. 企业改制，特别是产权制度

个人审批的分管领导：总经济师、总工程师、总会计师、总审计师、总法律顾问等

何种形式决定或审批

  • 决议（集体）
  • 审批意见（个人）

审批权限如何设置

  • 法律法规（公司法与三重一大决策制度等）［三重一大一定要集体决策］
  • 公司类型（国企、上市公司与私企等）与公司地位（母公司、子公司与分公司）［母公司的审批作为前置性程序，同意了才能提交子公司董事会（因为董事会批过了就有法律效力了）］
  • 计划或预算（内外或超计划预算）
  • 业务或事项的重要性（定性与定量）

三重一大

  • 重大决策事项
  1. 企业发展战略、破产、改制、兼并重组、资产调整、产权转让、对外投资、利益调配、机构调整等方面的重大决策;
  2. 企业党的建设和安全稳定的重大决策。
  • 重大项目安排事项
  1. 年度投资计划
  2. 融资、担保项目
  3. 期权、期货等金融衍生业务项
  4. 重要设备和技术引进
  5. 采购大宗物资和购买服务
  6. 重大工程建项目。
  • 重要人事任免事项
  1. 中层以上干部和所属单位领导班子成员的任免、聘用、解除聘用和后备人选的确定；
  2. 向控股和参股企业委派股东代表，推荐董事会、监事会成员和经理、财务负责人。
  • 大额度资金运作事项
  1. 年度预算内大额度资金调动和使用;
  2. 超预算的资金调动和使用;
  3. 对外大额捐赠和赞助。

业务审批与财务审批

  • 两位分管领导的重复审批问题
  • 前置性审批对后置性审批权限设置的影响

决的时间要求

  • 流程中应规定时间

（4）附件程序

评审、论证

  • 常设的专门委员会（董事会下属的和总经理办公会下属的专委会）
  • 临时性的专家委员会（前期论证性的和后期验收性的）

鉴定

  • 技术性鉴定（技术先进性、资产报废等）
  • 合规性鉴定（司法鉴定等）
  • 真实性鉴定（身份和单证等）

审计

  • 先审计后提交董事会，一共两个审计报告提交（年报和内控报告）
  • 三种审计，管理审计、业务审计和财务审计
  • 管理审计（管理的工具箱的工具要审计）：战略审计、预算审计（行政事业单位七八月开始编，企业九月开始编）、内控审计、绩效审计等
  • 业务审计：工程审计（全过程跟踪审计）、教学审计、科研审计、安全管理审计、社会服务审计等
  • 财务审计：报表审计和财务收支审计

（5）需要设计哪些流程

全覆盖（内控的特征）
业务类流程：研发、设计、采购、存货、制造、质量、营销、售后服务、工程、外包、招投标、安全、

  • 工程建设招投标

公开招标（一般标的物超过30万就要公开招标）

  1. 找招标代理机构（招投标平台、中介）
  2. 提出代理需求，招标和中标的基本条件（招标需求文件）
  3. 招标代理机构的具体实施：编写具体的招标文件、发标、投标、接标、在规定时间开标、评标、定标、清标
  4. 报给招投标单位，单位进行复核（代理机构违规时有权推翻）

邀请招标

财务类流程：融资、担保、投资、资金、财务报告、资产、成本费用、税收事项等。

管理类流程：战略、组织架构、流程管理、风险评估与报告、合同、法律事务、信息系统、企业文化等。

  • 合同

从法律角度，一般要件（主体、客体、时间、权利与义务等）齐全
从风控角度，着眼于签订、管理的流程

  1. 合同签订：合同尽调、拟定草案、合同谈判、修改合同、相关部门会审会签（至少有业务部门、法务部门和财务部门）、法定代表人（或其授权人）签字
  2. 合同履行
  3. 合同变更
  4. 合同总结（反馈），履行情况如何
  5. 合同分析
  6. 合同考核

人资类流程：人力资源、岗位管理、绩效管理、薪酬管理、工会等。

  • 薪酬管理包括：薪酬方案的制定（要过董事会）和薪酬发放的流程

（二）内控之“神”：制衡

制衡机制：流程式制衡、竞争式制衡、分离式制衡、合作式制衡、审批式制衡、集中式制衡、信息式制衡、监督式制衡、问责式制衡、系统式制衡。

竞争式制衡

  • 外部竞争：市场竞争（招标等）
  • 内部竞争：岗位竞争等［任用要用竞聘制，高管少，中层（部门负责人）多］

流程式制衡

  • 设计制衡性的流程（提审决）

分离式制衡

  • 不相容职务的分离：权能分离（三权分离）和性态分离（三流分离）

权能分离：

  • 治理层：决策（董事会）、执行（总经理办公会）、监督（监事会）
  • 中间层：管理（法务、行政、财务）、经办（业务部门）、监督（内审、纪检监察等）
  • 操作层面：管理、操作、检查［联系具体岗位］

性态分离：钱、账、物

  • 分离方式：机构分设与岗位分设；横向分离与纵向分离
  • 三个角度：流程、功能、性态
  • 设立不相容岗位时考虑成本效益原则和重要性原则

合作式制衡

  • 会审：常用于合同和重要事项决策
  • 会签：一般指文件，文件发布前会签
  • 会议：主要关注董事会、总经理办公会、各种类型的专门委员会议（专委会）、各种专项工作会议（供应、生产、销售等）、职代会、监事会、党委会
  • 共同决策
  • 共同行动

“三重一大”集体决策就是通过会议、通过共同决策实现合作式制衡

审批式制衡

  • 集体审批与个人审批
  • 财务审批与业务审批

集中式制衡

  • 全口径集中，集团式集中：物流(集中采购)、信息流(集中核算)、资金流(财务共享)和人流(集中管理)
  • 功能式集中：归口集中管理、分散经办。

资金的管理方式：

  1. 分散管理：子公司的钱子公司自己管
  2. 集中管理，“资金池”模式，大势所趋
    • 内部成立资金集中/财务共享中心等
    • 成立财务公司（独立法人，过去人行批，现在银监会批）：由集团成立，在集团内部负责资金融通的非银行金融企业（吸收发放存贷款）［钱主要来自集团内部的和同业拆借］
    • 公司银行：金融资本和产业资本结合形成财团，可以向社会吸收发放存贷款（财务公司不能面向社会）

物流的管理方式

  1. 全口径集中，流程：列计划、给总公司申请、总公司批后转给采购中心、采购中心直接配送
    • 要求：信息系统、发达的物流体系、企业的组织形式是分公司而不是子公司
  2. 大宗物资采购集中：两类物资，金额较大价值较高的和批量较大的

信息流的管理方式

  • 三大信息流（三大核算）：业务核算、会计核算、统计核算
  • 会计通过财务共享集中核算，通过集中来实现共享

    集中的事项范围：资金集中、开户集中、核算集中、税务集中、票据集中（发票和银行的结算凭证）、人员集中

信息式制衡

  • 公开透明
  • 信息披露

监督式制衡

  • 内审
  • 监事会
  • 纪检监察
  • 监督式委派：产权代表和财务负责人

委派产权代表：委派董事（股权代表）和监事
委派财务负责人：

  • 人员种类：财务总监；财务负责人、财会机构负责人；一般会计人员委派
  • 薪酬方案多为派驻单位承担，委派单位统一发放
  • 功能类型：
    • 监督型，不参与决策管理（如政府向企业委派）
    • 扮演财务负责人角色（即监督、管理，更常见）
    • 监督+管理型
  • 委派类型
    • 直接委派（母公司人员到子公司去）
    • 身份委派（母公司赋予子公司人员新的身份）

问责式制衡

系统式制衡

  • 信息系统固化流程
  • 控制权力弹性

（三）内控目标：风险防控

风险评估与风险库

  • 整体风险（战略风险、市场风险、运营风险、财务风险、法律风险等）和业务风险
  • 风险等级（三级）
  • 风险评估流程：识别—分析—评价—报告

风险态度与风险应对

  • 风险态度：风险最小化与可接受风险水平
  • 风险应对：策略

内控与风险管理的关系：内控是风险管理的组成部分

（1）风险等级划分

（2）风险评估流程

风险：对目标实现造成不利影响的因素和计算（风险是针对目标而存在的）

信息搜集 -> 信息搜集表

  1. 宏观与微观
  2. 内部与外部
  3. 财务与非财务等

风险识别 -> 风险描述

  1. 风险名称。采用“目标或业务+风险”的描述方式。如财务风险、投资风险等。
  2. 风险描述。对风险发生的各种成因及其表现分类描述。

风险分析

  1. 风险发生可能性分析标准。分极高、高、中、低、极低五个等级；
  2. 风险影响程度分析标准。分极高、高、偏低三个等级。

风险评价 -> 风险评价矩阵

  1. 风险分为重大风险、重要风险和一般风险三个等级，划分的标准是按照风险影响程度和风险发生的可能性的组合。

风险应对

  1. 策略：风险承担、风险控制、风险规避、风险转移、风险转换、风险对冲、风险补偿等。

风险报告 -> 风险库

  1. 风险内容。包括风险名称、风险类型、风险描述等；
  2. 风险等级。包括重大风险、重要风险和一般风险三级；
  3. 风险应对。包括应对策略、主要措施等。

风险预警 -> 预警报告

  1. 预警指标体系的设计

担保不能授权，不管金额大小都要提交董事会和股东会

（四）内控载体

内控载体形式：内控矩阵式、制度+流程图式、图表式

  • 流程：谁做？做什么（控制活动）？依据什么做（输入）？做的结果（输出）？做的时间要求？
  • 制度：(1)覆盖内控要素； (2)突出风险防控主题； (3)把握内部制衡主线。
  • 表单：表格、单证、报告
  • 备注：制度形式：章节条款式、数字排列式、图表式

三、内控流程的分类分析

四流：资金流、业务流或物流、信息流、工具流。

  • 资金流：流入、流出、存量等 -> 财务控制。
  • 业务流或物流：采购、使用、存量 -> 存货和固定资产 -> 业务控制。
  • 信息流：业务信息、财务信息、统计信息 -> 信息系统 -> 会计控制。
  • 工具流：战略、预算、合同、考核等等 -> 管理控制。

战略

  1. 财务是战略的组成部分（财务战略和业务战略）
  2. 会计是战略的工具（战略分析需要会计、战略落地需要会计将战略目标分解）

合同

  1. （部分企业）合同归口财务管
  2. 财务部门参与合同谈判评审签订

绩效考核

  1. 财务本身就是绩效考核的一部分（平衡积分法/kpi等，国务院考核国企关注营业收入、经济增加值和净利润）
  2. 会计报表是考核的重要依据

（一）资金流控制

资金流控制内容：资金流量控制与资金存量控制。

资金管理模式：

  1. 分散管理：账户分散、存款分散、收支分散。
  2. 集中管理
  • 谁集中：母公司。
  • 集中方式：内部资金中心、财务公司与企业银行——资金池模式。
  • 集中内容：开户集中、存款集中、收支集中、核算集中。
  • 集中好处：整合资金、降低成本、增进管控。

资金流入控制

经营资金流入控制：应收账款控制——全程信用管理

  • 事前：信用政策设计、客户选择
  • 信用政策：分松紧两种类型，要素包括：
    • 信用期限
    • 信用额度（总额度&单项额度）
    • 信用条件（担保等）
    • 利息附加条件等
  • 客户选择：考虑
    • 收益（创造价值）
    • 成本（信用政策、融资成本）
    • 风险（坏帐损失）
    • 关系（市场契约型、集团内部股东关系、战略联盟）等
  • 事中：合同、账款回笼、核算
  • 合同：
    • 合同尽调
    • 合同签订（八大要素和特殊要素全不全，签订权限）
    • 流程（会审会签、专家评审、技术鉴定等，归口部门要定期对合同履行情况进行监督检查）
    • 合同变更等
  • 账款回笼： 现金折扣等
  • 事后：分析与考核
  • 分析重点：账龄、客户
  • 考核：业务部门（中国常见收款）、财务部门（贷款回笼主要责任人）、法务部门（不一定）

借款资金流入控制：借款计划控制、授信额度控制、借款额度控制、负债比例控制、单笔借款控制、借款流程与审批控制、担保控制。

金融性负债三大种类

  1. 金融机构借款：长期借款和短期借款
  2. 发行债券：应付债券
  3. （从其他单位）拆借资金：（金融行业同业拆借）同业拆借、（非金融行业其他单位借款）其他应付款
  • 授信额度控制：
  • 总授信额度，通过资产负债率确定
  • 单项额度，每家金融机构的授信额度，加起来不会超过总信用额度
  • 负债比例控制
  • 资产负债率
  • 短期负债率，越高风险越大
  • 金融负债率，越高风险越大
  • 适度的负债规模综合考虑
  • [宏观]经济运行
  • [中观]行业性质（第三产业>第二产业>第一产业，主要因为现金流转速度）
  • [微观]企业效益（盈利水平）
  • 借款流程
  • 财务部门提出
  • 长期借款董事会审批（不管金额大小）
  • 短期借款董事长或授权财务总监审批
  • 担保：董事会和股东会

投资资金流入控制：固定资产处置流程、无形资产处置流程、对外投资流程———论证与审批

非经营性资金流入控制：捐赠收入控制、政府补助控制。

资金流出控制

经营资金流出控制：采购付款控制流程、常规性费用支付流程、薪酬支付流程、税金支付流程——后置性审批授权

借款资金流出控制：还款计划控制、偿债审批控制。

投资资金流出控制：固定资产购置流程、无形资产购置流程、对外投资流程。

  • 流程主要节点
  • 计划：投资计划必须经董事会批准（年度投资计划、年度经营计划）
  • 可行性研究报告（重要控制点）：技术可行性（样品->产品->商品）、市场可行性、财务可行性（财务评价）、法律可行性、社会可行性、环境可行性、道德可行性等。投资失败一般问题是出现在可行性研究上，对市场过于乐观或财务评价只简单的计算了现金流未考虑影响现金流的因素
  • 立项
  • 合同
  • 实施控制
  • 记录
  • 收益
  • 处置

非经营性资金流出控制——捐赠支出控制

资金存量控制

  1. 库存现金控制：使用范围控制、库存额度控制、使用审批控制、职工借款控

  2. 银行存款控制：开户控制、额度控制、对账控制、网银控制或单子支付控制等。

  3. 单位卡控制

  4. 资金调度等。

（二）物流控制

物流管理模式选择

  1. 分散管理
  2. 集中管理、分散采购和销售
  3. 集中采购和销售。

采购管理：计划—申请与审批—招标—合同—采购—付款—供应商管理

  • 采购计划的编制（属经营计划的组成部分）：总经理办公会批
  • 申请采购（请购）：总经理或副总经理（分管领导）批
  • 招标选择供应商：公开招标或邀请招标
  • 付款：四个根据（合同、发票、相关证明和相关审批）
  • 供应商管理：名录、定期评价等

销售管理：计划—宣传—合同—发货—结算—核算。

存货管理：计划—验收与计量—库存（含安全库存）—发货—核算。

工程项目：

  • 项目管理
  • 成本管理：概算、预算、决算审计
  • 资金管理：预付款、进度付款、尾款

（三）信息流控制

信息流：会计、统计与业务。

信息流管理模式：

  • 分散核算
  • 集中核算（财务共享）
  • 结合模式：财务负责人委派与轮岗

集中的弊端：财务与业务脱节；业务缺乏财务支持、弱化财务的决策支持作用。

财务报告管理：会计裁量权—编制—审核—审计—审批—披露——分析与使用——保存档案

  • 会计裁量权：会计政策、会计估计的选择权，重要的董事会批准，一般的财务负责人决定
  • 编制：年度报告要有编制方案，包括编制原则、编制内容、编制范围、编制时间、编制流程、编制方法，设计的特殊项目、报送传递等
  • 审核：财务负责人，财会机构负责人，单位负责人
  • 审批：月报董事长批，季报中报年度报告董事会批，年报还要提交股东会
  • 披露：合规性和即时性
  • 分析与使用：虚资产的影响（长期待摊、递延所得税、待处理投资损益等）

信息系统管理：规划—方案—供应商选择—外包监控——验收——使用与维护——安全管理——信息管理。

（四）工具流控制

战略：编制方案—编制草案—审核—审定—论证—审批—执行—评估与调整——总结与分析。

预算：编制方案—编制草案—审核—审定—论证—审批—执行—调整—总结与分析。

风控

绩效方案的编制与审批——考核实施及结果确认——结果运用——绩效改进。
