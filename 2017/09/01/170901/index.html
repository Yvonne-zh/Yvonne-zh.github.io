《一百个人的十年（冯骥才）》书摘

发表于 2017-09-01 | 分类于 读书
从土里长出的历史，从土里长出的民族。

崇拜是把自己掏空了，交给人家。如果人家拿过去随手一扔，或在人家手里丢失了，你呢?你就光剩下一个空壳，整个完了！人生是一次性的。你便永远像个空纸盒那样被遗落在世上，无法挽回。

拯救灾难的，不是圣贤，永远是人民

政治从来不对人的心灵负责。

真正的残暴，是针对无辜。

嘛叫“文革”的特点？它总叫你以为，当时那样就是永远那样。你要真的这样认为，错了！儍小子，“文革”就是不停止的翻来覆去，你上我下，你死我活，你喜我悲。

我只能说——如果世界上还有比原于弹更厉害的东西，那就是“文化大革命”！

到现在只能给街道副食店干点会计。在家闲得难受着呢。街道问我：“你干得了吗?”我说：“当年铁路施工预算我都搞，这么简单的玩艺还不行。我是高级工程师呀！”他们都笑起来。我也不知道他们笑的什么。

人都是正常的，荒诞都是生活的强加。换句话说，荒诞是生活的本质。

实际上我现在认识到，我们学校的这个“文革”是怎么回事呢，学生的这个革命跟这个老师宗派的派性斗争，搅在一块啦就是，而我们呢，就成了人家利用的工具。可又身不由己，整个时代往前发展也许只能那样做了。你上了台你就下不来了。

所谓罪状大多是闲事。“文革”批人，都是把正事和闲事相结台。中国人哪越是闲事大家越感兴趣，越能搞臭一个人。

我就想，地球那么大，天天你跟我斗，我跟你斗，有什么劲呀！不累吗？

“文革”就是没能耐的整有能耐的。他们没能耐，不靠这机会，不把咱整下去，他们就上不来。“四人帮”离老百姓远着哩。

村长这位置很关键，谁把住这位置谁就好压对方，不叫对方出头。但姓王的人多，势力大，一直占着村长。敌物时期保甲长是他们，后来八路军来了，村干部又大都姓王。从这里，你可以看到历史怎样变成现实，要寻找文革的渊源，这便是最长最远最深的一条根了。可是这道理我当时怎么可能认识到？

文革期间，一个造反派对我说，你知道你的档材料有多少吗？一车也拉不完！对我这样耿耿忠于革命的人，怎么花费这么大力气来监视、控制、调查呢？我想不通，上了强劲。第二你敢说，你在自己的档案里是个好人吗？如果你认为自己赤胆忠心，忠于国家，忠于事业，你感到你的心、你的血液、你整个身体都是透明的。但夹在那厚厚的硬紙封皮的档案中的你，却可能是满身污点的另一个人。关键时刻人家都是从这“另一个”的你来判断你，对待你，决定你。

“文革”过去，家破的家破，人亡的人亡，所有债都记在“四人帮”身上，人对人又是笑脸，又都一个样儿了，哪里去找当初那些整人的人？除非他们自己有良心，可是我从来没听说过有谁，天良发现。

从“四清”到“文革”，我看透了——中国没有真正搞科学的地方，处处，人人都槁政治。但不是政治家，是小政客们，政治小应声虫们。又不是真正搞政治，而是搞整人，互相整。今天你上来我下去，明天我上去你下来。整成一团团，谁也解不开，愈整愈带劲。

尽管文革被政治处死，它的幽灵犹存未泯。只要产生文革的土壤未被铲除，谁也无法保证文革永不再来。作为权力生命的文革已经消亡，但作为社会生命和文化生命的文革依旧额强地活。

# 书摘
《中国历代政治得失（钱穆）》书摘
《私人生活的变革：一个中国村庄里的爱情、家庭与亲密关系：1949～1999（阎云翔）》书摘
